# 🌐 הוספת שם נורמלי לאתר

## אפשרות 1: שם מותאם ב-Vercel (חינם)

כשאתה מפרס ב-Vercel, אתה יכול לשנות את השם של הפרויקט:

1. **לך ל-Vercel Dashboard**
2. **בחר את הפרויקט**
3. **לך ל-Settings > General**
4. **שנה את Project Name** לשם שאתה רוצה (למשל: `my-shopping-list`)
5. **השם החדש יהיה**: `my-shopping-list.vercel.app`

**הערה**: השם צריך להיות:
- באותיות קטנות
- ללא רווחים
- יכול להכיל מקפים ומספרים
- צריך להיות ייחודי (אם מישהו כבר לקח את השם, תצטרך לבחור אחר)

## אפשרות 2: Domain מותאם אישית (חינם או בתשלום)

### א. Domain חינמי מ-Freenom

1. **צור domain חינמי**:
   - לך ל: https://www.freenom.com
   - חפש domain (למשל: `mygroceries.tk` או `mygroceries.ml`)
   - הוסף לעגלה והשלם (0$)

2. **הגדר ב-Vercel**:
   - לך ל-Vercel Dashboard > הפרויקט שלך
   - Settings > Domains
   - הוסף את ה-domain שלך
   - עקוב אחר ההוראות להגדרת DNS

### ב. Domain בתשלום (מומלץ)

1. **קנה domain**:
   - Namecheap: https://www.namecheap.com
   - GoDaddy: https://www.godaddy.com
   - Google Domains: https://domains.google

2. **הגדר ב-Vercel**:
   - לך ל-Vercel Dashboard > הפרויקט שלך
   - Settings > Domains
   - הוסף את ה-domain שלך
   - עקוב אחר ההוראות להגדרת DNS

## שלבים מהירים

### לשם מותאם ב-Vercel:

1. פרס את האתר ב-Vercel
2. לך ל-Settings > General
3. שנה את Project Name
4. שמור

### ל-Domain מותאם:

1. קנה/צור domain
2. לך ל-Vercel > Settings > Domains
3. הוסף domain
4. הגדר DNS לפי ההוראות

## הערות חשובות

- **Vercel מספק HTTPS אוטומטית** - לא צריך לעשות כלום
- **Domain מותאם חינמי** - עובד מצוין, אבל יכול להיות איטי יותר
- **Domain בתשלום** - בדרך כלל מהיר יותר ואמין יותר

## המלצה

להתחלה, פשוט **שנה את שם הפרויקט ב-Vercel** - זה הכי קל ומהיר!
